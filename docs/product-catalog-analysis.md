## 프레임 유형 (주문 시 선택)

5가지 기본 프레임 형태 제공:

```
┌─┐  ┌──┐  ┌───┐  ┌────┐  ┌─────┐
│ │  │  │  │   │  │    │  │     │
└─┘  └──┘  └───┘  └────┘  └─────┘
1칸   2칸    3칸    4칸     5칸
```

- 공간 크기에 맞춰 칸 수 선택
- 천장 높이에 맞게 커스텀 가능
