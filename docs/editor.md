  프로젝트명: 사용자 맞춤형 옷장 입면도 및 평면도 설계 도구 (Closet Design Tool)

1. 프로젝트 개요

사용자가 평면도(Top-view)에서 옷장의 전체 크기와 유닛 배치를 설정하면, 이를 기반으로 입면도(Front-view)가 자동 생성되고, 입면도 내부에 선반, 서랍 등 프리셋 부품을 드래그 앤 드랍으로 커스텀 설계할 수 있는 웹 기반 도구 개발.

2. 핵심 워크플로우 (Core Workflow)

[평면도 단계]: 벽면 길이를 설정하고 800mm, 1000mm 등 가구 유닛(프리셋)을 배치.

[자동 투영]: 평면에 배치된 유닛의 개수와 너비에 맞춰 입면도 상에 편집 가능한 영역(Slot) 자동 생성.

[입면도 단계]: 생성된 영역 안에 내부 부품(서랍, 옷걸이봉, 선반)을 드래그하여 배치. (영역 내 스냅 기능 포함)

[출력]: 완성된 평면도와 입면도를 이미지(PNG) 또는 설계 데이터(JSON)로 저장.

3. 상세 기능 요구사항

3.1 평면도 디자인 (Plan View Editor)

공간 정의: 옷장이 설치될 벽면의 전체 너비($W$)와 깊이($D$) 설정 기능.

유닛 배치:

미리 정의된 가구 모듈(600, 800, 1000mm 등)을 리스트에서 선택하여 배치.

유닛 간 자석 스냅(Snapping) 기능으로 틈새 없이 밀착 배치.

배치된 유닛의 순서 변경 및 삭제 가능.

데이터 연동: 평면도에서 배치된 유닛의 $x$좌표와 $width$ 값은 입면도 모델의 기본 구조가 됨.

3.2 입면도 디자인 (Elevation View Editor)

영역(Slot) 생성: 평면도 유닛 데이터를 기반으로 입면 가이드라인 자동 렌더링.

프리셋 컴포넌트 드랍:

선반(Shelf), 서랍(Drawer), 옷걸이봉(Hanging Rod), 빈 칸 등 부품 리스트 제공.

드래그 앤 드랍 시 해당 유닛 영역 내부로 '쏙' 들어가는 스냅 기능.

상세 조절:

배치된 부품의 높이($y$ 위치)를 마우스로 조절.

부품 클릭 시 상세 치수(mm) 표시 및 입력 수정 기능.

3.3 이미지 및 데이터 출력

이미지 캡처: 현재 캔버스(평면/입면)를 고해상도 PNG 이미지로 내보내기.

통합 뷰: 평면도와 입면도를 한 장의 도면 양식으로 합쳐서 내보내는 기능.

데이터 저장: 설계된 구조를 JSON 형태로 로컬 저장 및 불러오기.

4. 기술 스택 및 구현 가이드라인

Framework: React.js (상태 관리가 용이함)

Canvas Library: Konva.js 또는 Fabric.js (드래그 앤 드랍 및 객체 조절 최적화)

Styling: Tailwind CSS (깔끔한 UI/UX 구현)

Data Model:

모든 설계는 단일 소스(JSON Object)로 관리되어야 함.

유닛 내부에 children 배열을 두어 입면 부품들의 위치값을 저장.

5. UI/UX 디자인 가이드

화면 분할: 왼쪽은 컴포넌트 라이브러리(사전 정의된 가구/부품), 중앙은 메인 워크스페이스(Canvas), 오른쪽은 속성 편집기.

인터랙션: 부품을 드래그할 때 들어갈 수 있는 영역을 하이라이트 표시하여 사용자 경험 극대화.

반응형: 모바일 기기에서도 기본적인 확인 및 간단한 수정이 가능하도록 fluid layout 적용.

6. 예외 처리

유닛의 너비를 벗어나는 부품 배치 방지 로직.

부품끼리 수직으로 겹칠 때 자동 정렬 또는 경고 표시.

잘못된 배치 시 되돌리기(Undo/Redo) 기능 포함.
